Endpoints

Add a profile | request type: POST

Description: Adds the profile to the table if said profile exists.

If the profile is private, a [PRIVATE] tag is added to the username.

localhost/api/v1/profiles/add?username=[USERNAME]

Get profile | request type: GET

Description: Returns a PSN profile object, without adding it to the table.

Example:
                {"id":0,
                "username":"XxGamer88 ",
                "level":338,
                "levelProgress":"3%",
                "totalTrophies":2897,
                "platinumTrophies":51,
                "goldTrophies":356,
                "silverTrophies":662,
                "bronzeTrophies":1828}

localhost/api/v1/profiles/getPSNProfile?username=[USERNAME]

Get all profiles | request type: GET

Description: Returns all PSN profile objects from the table table.

localhost/api/v1/profiles/getall

Delete profile | request type: DELETE

Removes a profile from the table if said profile exists.

localhost/api/v1/profiles/remove?username=[USERNAME]
